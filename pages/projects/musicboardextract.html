MusicboardExtract



Introduction

MusicboardExtract is a graphical user interface application for extracting data from MusicBoard, including lists and reviews.

The user can select the type of data they want to extract and the app will generate a csv file with the data.

Shows the extracting process.

It has a Webapp version

Technologies

This app was developed using:

  • Python
  • Tkinter
  • BeautifulSoup
  • Selenium
  • Vercel

The Webapp version is hosted on Vercel.
