Spotle Infinity



Introduction

Spotle Infinity is a little website that generates a random Spotle game for you to play.

The game Spotle is created by latwhite studios, and we would like to give them a shout out for their amazing game.

The website is solely for generating Spotle games and does not gather any user data.

It generates a graph accordingly and show the trainer all the necessary data

The app is responsive and can be used on mobile devices.

It has a login system to ensure only certified trainers can access it.
