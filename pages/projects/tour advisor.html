Tour Advisor



Introduction

TourAdvisor is a student project developed for a Kotlin module in a university course.

It is a mobile application that aims to provide a similar experience to TripAdvisor, focusing on helping users plan and discover trips.

The app allows users to search for places, read reviews, and plan their trips.

It also has a feature to create a trip and add places to it.

Technologies

This app was developed using:

  • Kotlin
  • Java
