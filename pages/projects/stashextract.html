Stash Extract



Introduction

StashExtract_GUI is a graphical user interface application for extracting data from stash, including lists and reviews.

Technologies

This app was developed using:

  • Python
  • BeautifulSoup
  • selenium
  • tkinter

Next up

The next steps are to implement a database with all the data and not only the login data.

This way the trainer can keep track of all the clients' progress.

We want to have a feature to export the data to a csv file and not only to an image like the current version.

The app will also have a feature to send the data to the client's email.
