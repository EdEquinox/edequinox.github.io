strokey



Introduction

The app allows users to use a MIDI instrument or interface to trigger actions on their PC.

It features a graphical user interface (GUI) and is highly configurable.

The user can set the key, the action, and the MIDI note to trigger the action.

The app also has a feature to show the user the MIDI notes being played.

Technologies

This app was developed using:

  • C++
  • JUCE
