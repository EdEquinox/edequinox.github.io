Lactometer



Introduction

This apps helps trainers to keep track of their clients' progress.

The trainer inserts the data:

  • Velocity (km/h or W)
  • Heart rate (bpm)
  • Lactate (mmol/L)

and the app calculates the lactate threshold and the training zones.

It generates a graph accordingly and show the trainer all the necessary data

The app is responsive and can be used on mobile devices.

It has a login system to ensure only certified trainers can access it.

Technologies

This app was developed using:

  • Django
  • Html
  • Css
  • Python
  • Vercel

The app is hosted on Vercel and the database is Postgres.

Next up
